# Portfolio Website

A modern and responsive portfolio website designed to showcase my projects, technical skills, and professional experience as a frontend developer.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Project showcase with live demos and GitHub links
* Professional experience and education timeline
* Modern UI with glassmorphism effects and smooth animations
* Contact section for networking and opportunities

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* Vite

## Projects Featured

* ApsaraPOS – Khmer Point of Sale system built for a real business
* CelebrationHub – Personalized celebration page platform
* Additional web development projects and experiments

## Getting Started

```bash
npm install
npm run dev
```
